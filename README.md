## ⚡ 第一次歐洲自由行？先讀這段

如果你是第一次去西歐自由行，以下幾件事比任何景點都重要：

**🍽 西班牙的用餐時間跟你想的完全不一樣。** 午餐 14:00 才開始、晚餐 21:00 以後。中午 12 點去餐廳，廚房還沒開。入境隨俗，調整生理時鐘。

**🛡 扒手是真實存在的風險。** 巴黎地鐵、巴塞隆納蘭布拉大道、里斯本 28E 電車是三大高風險區。背包前背、手機不外露、護照留飯店保險箱。

**💳 幾乎全面刷卡。** 西班牙和葡萄牙刷卡普及程度甚至超過台灣，連路邊小攤都能刷。但格拉納達老城區的小 tapas 吧常只收現金，隨身帶 €50–80 備用。

**📱 eSIM 最方便。** 出發前買好歐洲 eSIM（Airalo 或 Holafly），到了自動啟用，16 天建議 10GB 以上。歐盟境內無漫遊費。

> 詳細實用資訊請開啟 `index.html` 點選「🧳 實用須知」分頁。

## 🌦 旅遊季節說明（11 月）

11 月為伊比利半島旅遊淡季，是文化歷史型旅遊者的最佳時機：

- 👥 人潮比夏季少 40–50%，熱門景點不需排隊 1–2 小時
- 🌡 氣候舒適（但溫差大）：巴黎 8–13°C、葡萄牙 14–20°C、安達魯西亞白天 18–22°C 夜間 5–12°C
- 💰 住宿與廉航票價比旺季便宜 30–40%
- 🎟 阿爾罕布拉宮等熱門票較容易搶到（但仍需提前）
- ⚠️ 注意：11月1日（日）為西葡法定假日 Todos los Santos，主要景點照常開放
- ⚠️ 日照短：11月日落約 17:30（南歐）至 17:00（巴黎），戶外景點提早安排

## 📂 互動式 HTML 功能說明

開啟 `index.html` 後：

- **城市篩選按鈕**：點選城市名稱，只顯示該城市的天數（交通段自動隱藏）
- **全部**：顯示完整 16 天行程與所有移動段
- **🚇 交通票券**：各城市地鐵路線圖連結、票卡/一日券/遊客卡比較、機場交通、跨城長途交通
- **🎫 訂票清單**：切換至按緊急程度排序的購票 Checklist，含景點官網連結
- **🧳 實用須知**：第一次歐洲自由行必讀——用餐時間、金錢付款、上網通訊、打包建議、安全防盜、交通卡省錢術
- **📍 Google Maps 連結**：所有景點、餐廳、市場等地點皆可點擊直接開啟 Google Maps 導航（共 77 個連結）
- **Dark Mode**：自動跟隨系統設定，支援深色模式
