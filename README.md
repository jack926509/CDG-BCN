## 🌦 旅遊季節說明（11 月）

11 月為伊比利半島旅遊淡季，是文化歷史型旅遊者的最佳時機：

- 👥 人潮比夏季少 40–50%，熱門景點不需排隊 1–2 小時
- 🌡 氣候舒適：葡萄牙 16–20°C、安達魯西亞 18–22°C
- 💰 住宿與廉航票價比旺季便宜 30–40%
- 🎟 阿爾罕布拉宮等熱門票較容易搶到（但仍需提前）
- ⚠️ 注意：11月1日（日）為西葡法定假日 Todos los Santos，主要景點照常開放

## 📂 互動式 HTML 功能說明
